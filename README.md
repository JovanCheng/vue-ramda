# vue-ramda
A wrapper for integrating [ramda](http://ramdajs.com/) to [Vuejs](http://vuejs.org)

## How to use it

```
npm install --save vue-ramda
```

Add to you main.js
```
import Vue from 'vue'
import VueRamda from 'vue-ramda'

Vue.use(VueRamda)
```

use it in `.vue` file:

```
this.$R.add(1, 2)
```
